# Security Policy

## Supported Versions

Security fixes are handled on the `main` branch.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report security concerns privately to the maintainer:

- LinkedIn: https://www.linkedin.com/in/sehastrajit-s/

Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix, if you have one

The maintainer will review the report and coordinate a fix before public disclosure when appropriate.
