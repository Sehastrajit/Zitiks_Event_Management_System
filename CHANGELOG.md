# Changelog

All notable changes to ZITIKS will be documented in this file.

This project follows a simple release format based on version tags such as `v0.1.0`.

## [Unreleased]

## [0.2.0] - 2026-05-04

### Added

- Public open-source documentation and contributor setup
- GitHub issue and pull request templates
- Frontend CI workflow for pull requests
- Dependabot configuration
- CODEOWNERS and release workflow
- MIT license, security policy, code of conduct, and contribution guide
- Groq-powered ZITIKS chatbot for browse and event detail pages
- Frontend-owned event and map API routes for one-project deployment
- Release notes and changelog support

### Changed

- Updated About page and footer for a public open-source presentation
- Made Sehastrajit Selvachandran the sole listed developer
- Updated package metadata for public open-source distribution

## [0.1.0] - 2026-05-03

### Added

- Next.js event browsing application
- Search, category, date, location, and price filters
- Event details pages with timeline and location sections
- Interactive event map
- Create-event interface
- Next.js API routes for event and map data
- Optional blockchain ticket booking integration
